== Description ==
VideoStories is Modern Designed and Clean Video, Movie related websites. It's perfectly suitable for Video Bloggers, Movie Review websites and Movie Marketers. VideoStories comes with Widgets Ready, Post Format Supports, Full width Template, Translation Ready and many other exciting features.

Our Support: http://prowptheme.com/support/

== Installation ==

1. In your WordPress Admin Panel, go to Appearance > Themes and click the Add New button.
2. Click Upload and Choose File, then select the videostories.zip file. Click Install Now.
3. Click Activate to use your new theme right away.
